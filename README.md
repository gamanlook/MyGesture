三指輕碰 = 滑鼠中鍵（可快速另開分頁）
三指下滑 = ESC（可快速關閉預覽）
四指輕碰 = 填滿視窗（可快速讓視窗最大化）

<br>

## 注意事項
- 觸控板設定：去系統設定裡，把會衝突打架的手勢（三指輕觸、三指下滑等）關掉。
- 給予權限：去「隱私權與安全性 → 輔助使用」把權限打開。
